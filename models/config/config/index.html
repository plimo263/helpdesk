    config_exists(**kwargs) -> bool

    Verifica se a configuração existe baseado no parametro.

    Parameters:

    Name Type Description Default
    id

    Consulta baseado no id

    		 required
    name

    Consulta baseado no nome da configuracao

    		 required

    Examples:

    get 

    get(**kwargs) -> ConfigData

    Recupera uma configuração baseado em um parametro nomeado enviado. Caso não seja encontrada a config uma Exception ValueError será lançada.

    Parameters:

    Name Type Description Default
    name

    O nome da configuração

    		 required
    id

    O identificador da configuração.

    		 required

    Examples:

    >>> c = Config()
>>> c.get(id=1)
ConfigData(id=1, name='TESTE', value='2', description='UMA DESCRICAO')

    get_all 

    get_all() -> Dict[str, ConfigData]

    Retorna todas as configs existentes no sistema como um Dicionario de configs.

    Examples:

    >>> c = Config()
>>> c.get_all()
{'TESTE': ConfigData(id=1, name='TESTE', value='2', description='UMA DESCRICAO')}

    save 

    save(config: ConfigData) -> ConfigData

    Cria uma nova configuração no sistema, caso o nome da configuração já exista uma Exception (ValueError) é lançada.

    Parameters:

    Name Type Description Default
    config ConfigData

    Um objeto de configuração para criação de uma nova config.

    		 required

    Examples:

    >>> c = Config()
>>> c_data = ConfigData(name='TESTE', value='1', description='UMA DESCRICAO')
>>> c.save(c_data)
ConfigData(id=1, name='TESTE', value='1', description='UMA DESCRICAO')

    update 

    update(config: ConfigData) -> ConfigData

    Atualiza uma configuração já existente, uma exception pode ser lançada caso o DB não aceite algum parametro enviado.

    Parameters:

    Name Type Description Default
    config ConfigData

    Um objeto de configuração para atualização de uma config existente.

    		 required

    Examples:

    >>> c = Config()
>>> c_data = c.get(1)
>>> c_data.value = '2'
>>> c.update(c_data)
ConfigData(id=1, name='TESTE', value='2', description='UMA DESCRICAO')
