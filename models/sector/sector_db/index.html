    Class SectorDB

    Classe para lidar com atividades relacionadas ao setor.

    __upsave_sector_db 

    __upsave_sector_db(sector: SectorTable) -> int

    Grava o setor e/ou atualiza ele no banco de dados. Lança uma Exception caso não consiga realizar a gravação.

    Parameters:

    Name Type Description Default
    sector Sector

    Uma instancia do ORM SectorTable para cadastro/atualização do setor

    		 required

    Examples:

    >>> s_reg = SectorTable.query.get(1)
>>> s_reg.nome = 'Recursos Humanos'
>>> s = SectorDB()
>>> s.__upsave_sector_db(s_reg)
1
